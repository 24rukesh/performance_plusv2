## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Build-time `next/font/google` fetch → IBM Plex woff2 files | Next.js downloads font files from Google at build time and self-hosts them. Standard supply chain. |
| Heroicons SVG paths → DOM render | Hardcoded SVG paths from a Tailwind Labs MIT package. No user-controlled SVG content. |
| Hardcoded demo card data → React render → DOM | All 4 demo card strings (campaign_id, label, reasoning, pct) are source literals. No external input. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-15 | Tampering (XSS via React) | Hardcoded demo card data + feature copy + footer string rendered as JSX text | accept | React auto-escapes string children. No `dangerouslySetInnerHTML` used in any file in this plan (verified by build-time TypeScript check + grep gate). Inline style props on demo card badges set CSS only (no `__html`). |
| T-07-16 | Information Disclosure (font CDN fetch at build) | `next/font/google` fetches woff2 files from fonts.googleapis.com during `npm run build` | accept | Standard Next.js behavior; fonts are then self-hosted in the build output (no runtime CDN dependency). No PII transmitted. Same fonts already used by the Streamlit app via `<link>` — same supply chain. |
| T-07-17 | Tampering (animation timing manipulation) | IntersectionObserver disconnect after first intersection | mitigate | Observer disconnects on first `isIntersecting: true`, so the animation cannot be re-triggered via DOM manipulation. Animation is single-play (no `animation-iteration-count: infinite`). `animation-fill-mode: both` holds the final state. |
| T-07-18 | Denial of Service (animation re-trigger / memory leak) | IntersectionObserver cleanup in useEffect | mitigate | useEffect returns `() => observer.disconnect()` as cleanup, plus the observer self-disconnects on first intersection. No leaked observers if the component unmounts during scroll. |
| T-07-19 | Supply chain (Heroicons import) | `@heroicons/react/24/outline` | accept | Heroicons is maintained by Tailwind Labs (MIT). Tree-shaken at build — only the 4 imported icon symbols ship in the bundle. UI-SPEC permits inline SVG as a fallback if the dependency is later removed. |
