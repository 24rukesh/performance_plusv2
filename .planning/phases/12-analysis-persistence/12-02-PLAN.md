def run_analysis(campaign_agg: pd.DataFrame) -> AnalysisResult:

    if os.environ.get("DEMO_MODE") == "1" and client is None:

    t0 = time.perf_counter()
    csv_text = campaign_agg.to_csv(index=False)
    try:
        result = _call_llm(csv_text)

        logger.warning("OpenAI API error — falling back to fixture: %s", exc)

| llm.py → st.session_state | The except branch writes api_fallback_active=True to Streamlit session state. This must be guarded against non-Streamlit contexts (tests, CLI usage). |

| T-12-05 | Tampering | except openai.OpenAIError scope | mitigate | Catch only openai.OpenAIError base class — not Exception. Enforced by the parametrized test_value_error_does_not_trigger_fallback test. |
