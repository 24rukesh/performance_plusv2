From api/db.py (connection pattern — copy verbatim, drop the FastAPI HTTPException import):
```python
import os
import psycopg2
import psycopg2.errors
import psycopg2.extras  # required: Json adapter + JSONB auto-deserialization
from psycopg2.extras import Json

def get_conn():
    """Open a new connection. Caller is responsible for close()."""
    url = os.environ.get("DATABASE_URL")
    if not url:
        raise RuntimeError("DATABASE_URL is not configured")
    return psycopg2.connect(url)
```

From api/db.py (init_db double-context-manager pattern — use for every function in st_db.py):
```python
def init_db() -> None:
    conn = get_conn()
    try:
        with conn:
            with conn.cursor() as cur:
                cur.execute("""CREATE TABLE IF NOT EXISTS ...""")
    finally:
        conn.close()
```

From api/db.py (INSERT + RETURNING pattern for save_analysis):
```python
cur.execute(
    "INSERT INTO analysis_runs (label, payload) VALUES (%s, %s) RETURNING id",
    (label, Json(payload)),
)
row = cur.fetchone()
return row[0]   # new integer id
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| app.py → st_db.py | Analysis payload (dict containing AnalysisResult, DataFrames serialized as records) flows from Streamlit session state into Postgres JSONB. |
| Postgres → app.py | JSONB payload is deserialized by psycopg2 and passed back to Streamlit session state via load_analysis(). |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | Tampering | analysis_runs.payload JSONB | accept | No user-controlled input reaches the payload directly — it is built from st.session_state by app.py. Payload is not executable. Low risk for a single-user self-hosted app. |
| T-12-02 | Information Disclosure | DATABASE_URL env var | mitigate | DATABASE_URL read from environment variable only, never hardcoded. .env is gitignored. Docker compose injects from host env. |
| T-12-03 | Denial of Service | Unbounded JSONB payload size | accept | Payload is bounded by campaign_agg + merged_df serialized via to_dict(orient="records"). For realistic campaign counts (<100) this is well under 1 MB. Postgres JSONB limit is 256 MB — not reachable in normal use. |
| T-12-04 | Spoofing | label TEXT input | mitigate | Label is stored as TEXT, rendered via st.markdown in app.py. Use st.markdown(label) not st.markdown(label, unsafe_allow_html=True) to prevent XSS injection through label content. |
| T-12-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages installed in this plan — psycopg2 is already in pyproject.toml. Slopcheck not triggered. |
</threat_model>
